# Sada

A simple web application that allows users to receive anonymous messages through a personal link.

Overview

Sada is designed to let people share a link and receive honest, anonymous feedback or messages from others.
The experience focuses on simplicity, privacy, and ease of use.

---

How it works

- Each user gets a unique link
- The link can be shared with others
- Anyone with the link can send a message
- Messages are received privately by the owner of the link

---

Usage

1. Open the app
2. Sign in
3. Copy your personal link
4. Share it with others
5. Start receiving messages

---

Notes

- Messages are anonymous
- The app is intended for simple and direct communication
- No additional setup is required for basic usage
